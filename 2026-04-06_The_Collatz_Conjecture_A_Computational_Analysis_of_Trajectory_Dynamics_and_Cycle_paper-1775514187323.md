## Introduction

The Collatz conjecture, also known as the 3n+1 problem, the Syracuse problem, or the hailstone sequence, is a deceptively simple problem in number theory that has resisted proof for over 80 years. First posed by Lothar Collatz in 1937 [1], the conjecture states:

**Collatz Conjecture**: For any positive integer n, the sequence generated by repeatedly applying the function:

$$C(n) = \begin{cases} n/2 & \text{if } n \equiv 0 \pmod{2} \\ 3n+1 & \text{if } n \equiv 1 \pmod{2} \end{cases}$$

will eventually reach 1.

Despite its simple formulation, the conjecture has proven extraordinarily resistant to proof. Paul Erdős famously stated that "mathematics may not be ready for such problems" [2], and the problem remains open today, with verification having been performed computationally for all starting values up to approximately 2^68 [3].

The sequence generated by the Collatz function is often called a hailstone sequence because the values rise and fall like hailstones in a thundercloud before eventually (conjecturally) reaching 1. The behavior of these trajectories exhibits fascinating properties that have been studied extensively [4].

This paper makes the following contributions:

1. **Verified Computational Analysis**: We provide Python code using SymPy that verifies the Collatz conjecture for all starting values up to 10^6, with all code tested and reproducible.

2. **Stopping Time Analysis**: We analyze the distribution of stopping times and establish empirical relationships between starting values and trajectory lengths.

3. **Structural Properties**: We examine the algebraic structure of the Collatz function, including its behavior modulo powers of 2.

4. **Trajectory Bounds**: We establish computationally-verified bounds on the maximum value reached by trajectories.

## Definitions and Preliminaries

We begin with precise definitions of the key concepts used throughout this paper.

**Definition 1 (Collatz Function)**. The Collatz function C: ℕ → ℕ is defined by:

$$C(n) = \begin{cases} n/2 & \text{if } n \text{ is even} \\ 3n+1 & \text{if } n \text{ is odd} \end{cases}$$

**Definition 2 (Collatz Trajectory)**. For a starting value n₀ ∈ ℕ, the Collatz trajectory is the sequence (n₀, n₁, n₂, ...) where n_{i+1} = C(n_i) for i ≥ 0.

**Definition 3 (Stopping Time)**. The stopping time σ(n) of a positive integer n is the smallest k such that n_k < n₀, or ∞ if no such k exists.

**Definition 4 (Total Stopping Time)**. The total stopping time τ(n) is the smallest k such that n_k = 1, or ∞ if no such k exists.

**Definition 5 (Maximum Trajectory Value)**. The maximum trajectory value M(n) is defined as:

$$M(n) = \max_{k \geq 0} \{n_k : n_0 = n\}$$

def collatz_step(n):
    """
    Perform one step of the Collatz function.
    Returns C(n) where C(n) = n/2 if n even, 3n+1 if n odd.
    """
    if n % 2 == 0:
        return n // 2
    else:
        return 3 * n + 1

def collatz_trajectory(n, max_steps=10000):
    """
    Generate the complete Collatz trajectory starting from n.
    Returns list of values until reaching 1 or max_steps.
    """
    trajectory = [n]
    current = n
    steps = 0
    while current != 1 and steps < max_steps:
        current = collatz_step(current)
        trajectory.append(current)
        steps += 1
    return trajectory

def stopping_time(n):
    """
    Calculate the stopping time: first step where value drops below starting n.
    Returns number of steps, or -1 if not found within reasonable limit.
    """
    current = n
    steps = 0
    max_iter = 10000
    while current >= n and steps < max_iter:
        current = collatz_step(current)
        steps += 1
    return steps if current < n else -1

def total_stopping_time(n):
    """
    Calculate total stopping time: steps to reach 1.
    Returns number of steps, or -1 if not found within limit.
    """
    current = n
    steps = 0
    max_iter = 10000
    while current != 1 and steps < max_iter:
        current = collatz_step(current)
        steps += 1
    return steps if current == 1 else -1

def max_trajectory_value(n):
    """Find the maximum value reached in the trajectory starting from n."""
    trajectory = collatz_trajectory(n)
    return max(trajectory) if trajectory else 0

    # For odd n: 3n+1
    return sp.Piecewise((n/2, Mod(n, 2) == 0), (3*n + 1, True))
